## 序言

人类对世界的理解，始于区分——将混沌的经验切分为可辨认的单元。一片树叶、一次心跳、一轮日出。这种最原始的离散化，是一切认知与数学的起点。

《道德经》言“道生一，一生二，二生三，三生万物”，揭示了从不可分的整体到离散单元的生成过程。《易经》中“无平不陂，无往不复”则指出了宇宙运动的基本特征：一切变化都在周期与往复中展开。然而，这些古代的洞察长期停留在隐喻层面，未能获得形式化的表达。

周期数学的目标，是将这些关于离散生成与周期演化的直觉，用严格的数学语言重新表述。它不依赖于任何具体的文化符号或传统术语，而是从三个最基本的原则出发：

1. **宇宙是信息与信息关系演化的总和** ——信息场，而非物质或能量，是基本的实在。
2. **数学是人类认识宇宙的真子集** ——数学形式化地捕捉了信息关系中的可计算部分。
3. **周期是信息演化的普遍特征** ——信息场在梯度、散度、旋度的作用下，必然涌现出离散的周期结构。

本文档是周期数学的基础框架。我们从数的构造出发，经由向量与函数论，最终达到场及其基本算子。全篇不借助任何玄学或传统文化术语，仅依靠数学与物理学的公共语言。

## 方法论

周期数学的构造遵循四条铁律：

1. **认知发生优先**
   数学对象的定义顺序必须符合认知的逻辑：从最简单的离散区分开始，逐层构造更复杂的关系。不预设任何尚未构造的概念。
2. **无循环构造**
   每一步只使用已严格定义的对象。实数与微积分不得用于定义自然数或整数；完备性概念不得先于有理数出现。
3. **离散起点**
   可认知的世界始于离散区分——这是信息的最小单位。连续性是构造的产物，而非预设的基底。
4. **对称性即方向性**
   每一类新数都引入一种新的对称操作：整数引入加法逆（0对称），有理数引入乘法逆（1对称），复数引入相位旋转（单位圆对称），四元数引入三维非交换旋转。这些对称性赋予了数以“方向”的含义，使它们不仅表达大小，更表达在特定关系下的指向。

在方法论指导下，我们从皮亚诺公理开始，构造全部标准数系，进而发展向量与函数论，最终建立场论。

## 数系

### 定义

数是信息关系中稳定结构的符号化。数系是按对称性与完备性逐级扩展的代数系统。每一级扩展都引入一种新的关系类型，并通过严格的等价类构造或完备化实现。

### 自然数 $\mathbb{N}$

**公理（皮亚诺算术）**

1. $0$ 是自然数。
2. 每个自然数 $n$ 有唯一后继 $S(n)$，且 $S(n)$ 是自然数。
3. $0$ 不是任何自然数的后继。
4. 若 $S(n) = S(m)$，则 $n = m$。
5. 归纳公理：若性质 $P$ 满足 $P(0)$，且 $P(n)\Rightarrow P(S(n))$ 对所有 $n$ 成立，则 $P$ 对所有自然数成立。

**约定：** $1:=S(0),\;2:=S(1),\;\ldots$

**运算（递归定义）**
- 加法：$n+0:=n,\; n+S(m):=S(n+m)$
- 乘法：$n\times 0:=0,\; n\times S(m):=(n\times m)+n$
- 序：$n\leq m\iff \exists k\in\mathbb{N},\; n+k=m$

$(\mathbb{N},+,\times,\leq)$ 构成有序交换半环。自然数仅有“大小”，无逆元，无对称性。它是纯粹的标量，回答“有多少个离散元”。

### 整数 $\mathbb{Z}$

**构造** 在 $\mathbb{N}\times\mathbb{N}$ 上定义等价关系 $\sim$：
$(a,b)\sim(c,d) \iff a+d = b+c$

整数集 $\mathbb{Z}:=(\mathbb{N}\times\mathbb{N})/\sim$，等价类 $[(a,b)]$ 表示位移“从 $b$ 到 $a$”。

**运算**
- 加法：$[(a,b)]+[(c,d)]:=[(a+c,\;b+d)]$
- 零元：$0_{\mathbb{Z}}:=[(0,0)]$
- 加法逆：$-[(a,b)]:=[(b,a)]$
- 乘法：$[(a,b)]\times[(c,d)]:=[(ac+bd,\;ad+bc)]$
- 单位元：$1_{\mathbb{Z}}:=[(1,0)]$
- 序：$[(a,b)]\leq[(c,d)]\iff a+d\leq b+c$

$(\mathbb{Z},+,\times,\leq)$ 构成有序交换环。每个元素都有加法逆元，产生关于0的加法对称：对任意整数 $x$，存在唯一的 $-x$，使 $x+(-x)=0$。这一对称赋予整数以“位移方向”——正数表示一个方向的累积，负数表示相反方向的累积。整数可视为一维格点上的加法矢量（$\mathbb{Z}$-模意义下的向量）。自然数嵌入：$n\mapsto[(n,0)]$ 保持加乘结构。

### 有理数 $\mathbb{Q}$

**构造** 令 $\mathbb{Z}^\times:=\mathbb{Z}\setminus\{0\}$。在 $\mathbb{Z}\times\mathbb{Z}^\times$ 上定义 $\approx$：
$(a,b)\approx(c,d) \iff a\times d = b\times c$

有理数集 $\mathbb{Q}:=(\mathbb{Z}\times\mathbb{Z}^\times)/\approx$，等价类记为 $\frac{a}{b}$。

**运算**
- 加法：$\frac{a}{b}+\frac{c}{d}:=\frac{a\times d+b\times c}{b\times d}$
- 零元：$\frac{0}{1}$，加法逆：$-\frac{a}{b}:=\frac{-a}{b}$
- 乘法：$\frac{a}{b}\times\frac{c}{d}:=\frac{a\times c}{b\times d}$
- 单位元：$\frac{1}{1}$，乘法逆：若 $a\neq 0$，$(\frac{a}{b})^{-1}:=\frac{b}{a}$
- 序：$\frac{a}{b}>0\iff a\times b>0$，并由此推导全序。

$(\mathbb{Q},+,\times,\leq)$ 构成有序域。非零元素不仅有加法逆（0对称），还具有乘法逆（1对称）：对任意 $x\neq 0$，存在 $x^{-1}$ 使得 $x\times x^{-1}=1$。这一对称赋予有理数以“缩放方向”——$x$ 与 $x^{-1}$ 关于1对称，分别表示放大与缩小。整数嵌入：$n\mapsto\frac{n}{1}$。

1. $L\neq\emptyset$，$L\neq\mathbb{Q}$。
2. $L$ 没有最大元素。
3. 若 $x\in L$，$y\in\mathbb{Q}$ 且 $y<x$，则 $y\in L$（向下封闭）。

- 序：$n\leq m\iff \exists k\in\mathbb{N},\; n+k=m$

$(a,b)\sim(c,d) \iff a+d = b+c$

**运算**
- 加法：$[(a,b)]+[(c,d)]:=[(a+c,\;b+d)]$
- 零元：$0_{\mathbb{Z}}:=[(0,0)]$
- 加法逆：$-[(a,b)]:=[(b,a)]$
- 乘法：$[(a,b)]\times[(c,d)]:=[(ac+bd,\;ad+bc)]$
- 单位元：$1_{\mathbb{Z}}:=[(1,0)]$
- 序：$[(a,b)]\leq[(c,d)]\iff a+d\leq b+c$

$(\mathbb{Z},+,\times,\leq)$ 构成有序交换环。每个元素都有加法逆元，产生关于0的加法对称：对任意整数 $x$，存在唯一的 $-x$，使 $x+(-x)=0$。这一对称赋予整数以“位移方向”——正数表示一个方向的累积，负数表示相反方向的累积。整数可视为一维格点上的加法矢量（$\mathbb{Z}$-模意义下的向量）。

自然数嵌入：$n\mapsto[(n,0)]$ 保持加乘结构。

### 有理数 $\mathbb{Q}$

**构造** 令 $\mathbb{Z}^\times:=\mathbb{Z}\setminus\{0\}$。在 $\mathbb{Z}\times\mathbb{Z}^\times$ 上定义 $\approx$：
$(a,b)\approx(c,d) \iff a\times d = b\times c$

有理数集 $\mathbb{Q}:=(\mathbb{Z}\times\mathbb{Z}^\times)/\approx$，等价类记为 $\frac{a}{b}$。

**运算**
- 加法：$\frac{a}{b}+\frac{c}{d}:=\frac{a\times d+b\times c}{b\times d}$
- 零元：$\frac{0}{1}$，加法逆：$-\frac{a}{b}:=\frac{-a}{b}$
- 乘法：$\frac{a}{b}\times\frac{c}{d}:=\frac{a\times c}{b\times d}$
- 单位元：$\frac{1}{1}$，乘法逆：若 $a\neq 0$，$(\frac{a}{b})^{-1}:=\frac{b}{a}$
- 序：$\frac{a}{b}>0\iff a\times b>0$，并由此推导全序。

$(\mathbb{Q},+,\times,\leq)$ 构成有序域。非零元素不仅有加法逆（0对称），还具有乘法逆（1对称）：对任意 $x\neq 0$，存在 $x^{-1}$ 使得 $x\times x^{-1}=1$。这一对称赋予有理数以“缩放方向”——$x$ 与 $x^{-1}$ 关于1对称，分别表示放大与缩小。整数嵌入：$n\mapsto\frac{n}{1}$。

### 实数 $\mathbb{R}$

**构造（戴德金分割）** 实数集 $\mathbb{R}$ 由所有满足以下条件的下组 $L\subset\mathbb{Q}$ 构成：

1. $L\neq\emptyset$，$L\neq\mathbb{Q}$。
2. $L$ 没有最大元素。
3. 若 $x\in L$，$y\in\mathbb{Q}$ 且 $y<x$，则 $y\in L$（向下封闭）。

有理数 $q$ 对应下组 $\{x\in\mathbb{Q}\mid x<q\}$。

**运算**
- 序：$r\leq s\iff L_r\subseteq L_s$
- 加法：$L_{r+s}:=\{x+y\mid x\in L_r,\;y\in L_s\}$
- 乘法：先对正实数定义为 $\{xy\mid x\in L_r,y\in L_s,x,y>0\}\cup\mathbb{Q}_{\leq 0}$，再依符号扩展。

$\mathbb{R}$ 满足最小上界性质，即完备性。实数保留了加法逆和乘法逆（0对称与1对称），并引入连续性。它是连续场势的天然语言：每一点的强度、密度、变化率最终均由实数表达。实数本身是一维 $\mathbb{R}$-向量空间，其元素也是自身的“位置矢量”。

### 复数 $\mathbb{C}$

**构造** 复数集 $\mathbb{C}:=\mathbb{R}^2$。定义加法为逐分量相加，乘法为
$(a,b)\times(c,d):=(ac-bd,\;ad+bc)$

记 $a+bi:=(a,b)$，$i:=(0,1)$，则 $i^2=-1$。

**代数结构** $\mathbb{C}$ 是二维交换 $\mathbb{R}$-代数，且为域。每个非零复数有乘法逆 $(a+bi)^{-1}=\frac{a-bi}{a^2+b^2}$。

**极形式与旋转对称**
任意 $z\in\mathbb{C}$ 可写为 $z=re^{i\theta}$，其中 $r=|z|=\sqrt{a^2+b^2}$ 为模长，$\theta$ 为辐角。乘法 $z\mapsto e^{i\phi}z$ 保持模长不变，使辐角增加 $\phi$，即平面旋转。单位复数群 $\{e^{i\theta}\}$ 同构于 $U(1)$。复数在加法对称和乘法对称之外，引入了单位圆旋转对称：虚数单位 $i$ 是 $90^\circ$ 旋转算子。这使得复数成为平面内旋度效应的天然载体。

### 四元数 $\mathbb{H}$

**构造** 四元数集 $\mathbb{H}:=\mathbb{R}^4$，元素记作 $q=a+bi+cj+dk$。加法逐分量进行。乘法由以下规则双线性扩张：

- $i^2=j^2=k^2=ijk=-1$
- $ij=k,\; ji=-k$; $jk=i,\; kj=-i$; $ki=j,\; ik=-j$

乘法非交换，但满足结合律。$\mathbb{H}$ 是非交换 $\mathbb{R}$-除法代数。

**共轭与范数** $\bar{q}:=a-bi-cj-dk$，$|q|:=\sqrt{a^2+b^2+c^2+d^2}$。非零元的逆 $q^{-1}=\bar{q}/|q|^2$。

**旋转对称** 单位四元数群 $S^3$ 同构于 $SU(2)$，通过 $v\mapsto qvq^{-1}$（$v$ 为纯虚四元数）实现三维空间旋转。不同轴旋转的非交换性对应四元数乘法的非交换性。四元数在复数平面旋转的基础上，增添了完整的三维非交换旋转对称，是空间旋度结构的代数化身。

### 数系总览

| 数系 | 对称性 | 方向类型 |
| :--- | :--- | :--- |
| $\mathbb{N}$ | 无 | 纯大小 |
| $\mathbb{Z}$ | 加法对称（0对称） | 位移方向（正/负） |
| $\mathbb{Q}$ | 加法+乘法对称（1对称） | 缩放方向（放大/缩小） |
| $\mathbb{R}$ | 以上+连续完备性 | 连续强度与极限方向 |
| $\mathbb{C}$ | 以上+平面旋转对称 $U(1)$ | 相位旋转方向 |
| $\mathbb{H}$ | 以上+三维旋转对称 $SU(2)$ | 空间非交换旋转方向 |

## 向量

### 定义

向量是一组有序的数，用于表示具有大小和方向的量。根据分量所属的数系，向量可以建立在整数、有理数、实数、复数或四元数之上。向量空间的定义要求加法与标量乘法封闭，并且满足相应的公理。

- $\mathbb{Z}^n$：整数向量，构成格点（$\mathbb{Z}$-模）。
- $\mathbb{Q}^n$：有理数向量，构成 $\mathbb{Q}$-向量空间。
- $\mathbb{R}^n$：实数向量，标准的欧几里得向量空间。
- $\mathbb{C}^n$：复数向量，酉空间的基础，承载相位与旋转信息。
- $\mathbb{H}^n$：四元数向量，用于描述三维空间中的旋转与非交换变换。

向量的基本运算包括加法、标量乘法、内积、外积（叉积）等。不同数系上的向量拥有不同的代数结构与几何意义。只要不是自然数，任何数都拥有某种对称性，因而均可视为某种向量：整数有位移方向，有理数有缩放方向，复数有相位方向，四元数有三维旋转方向。

### 各类数向量的性质

| 向量类型 | 分量域 | 模长定义 | 方向表示 | 典型运算 | 物理对应 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 整数向量 | $\mathbb{Z}$ | 曼哈顿距离或格点范数 | 各分量符号与比值 | 加法、减法、标量乘（整数） | 晶格位移、离散步数 |
| 有理数向量 | $\mathbb{Q}$ | 欧几里得范数（平方根可能为无理数，但分量有理） | 各分量比值 | 加法、标量乘（有理数）、点积 | 比例缩放后的位置 |
| 实数向量 | $\mathbb{R}$ | $\sqrt{\sum x_i^2}$ | 与坐标轴的夹角 | 加法、标量乘、点积、叉积（三维） | 速度、力、场强 |
| 复数向量 | $\mathbb{C}$ | $\sqrt{\sum \|z_i\|^2}$ | 各分量的辐角与相对相位 | 加法、标量乘（复数）、内积（酉内积） | 电磁波振幅与相位、量子态 |
| 四元数向量 | $\mathbb{H}$ | $\sqrt{\sum \|q_i\|^2}$ | 各分量的旋转轴与角度 | 加法、标量乘（四元数）、非交换乘法 | 三维旋转、刚体姿态 |

### 向量关系与运算

以下定义向量之间及向量与标量之间的基本关系与运算。所有定义均基于纯数学与物理学含义，不涉及任何具体符号系统。

| 概念 | 数学含义 | 物理学含义 | 特性 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| 加法 | 向量逐分量相加 | 力的合成、位移叠加 | 交换律、结合律成立 | 平行四边形法则 |
| 减法 | 向量逐分量相减 | 相对位移、力的分解 | 加上逆向量 $a - b = a + (-b)$ | |
| 标量乘法 | 各分量乘以同一标量 | 缩放、强度调节 | 改变模长，保持或反转方向 | 若标量为负则反向 |
| 点积（内积） | $\vec{a}\cdot\vec{b} = \sum a_i b_i$ | 功、投影大小、能量耦合 | 交换律，结果为数 | 垂直时为零，同向时最大 |
| 投影 | $proj_{\vec{b}}\vec{a} = \frac{\vec{a}\cdot\vec{b}}{\vec{b}\cdot\vec{b}}\vec{b}$ | 将向量分解到给定方向上的分量 | 结果与 $\vec{b}$ 共线 | 垂直时投影为零向量 |
| 叉积（一维） | $a\times b := 0$ | 无旋转自由度 | 始终为零 | 一维无旋度 |
| 叉积（二维） | $z_1\times z_2 := \operatorname{Im}(\bar{z}_1 z_2) = ad - bc$ | 有向面积，二维涡旋强度 | 标量，反交换 | 正为逆时针，负为顺时针 |
| 叉积（三维） | $\vec{a}\times\vec{b}$，大小 $\|\vec{a}\|\|\vec{b}\|\sin\theta$，方向依右手定则 | 力矩、角动量、三维涡旋向量 | 反交换，结果为向量 | 三维旋度的完整形式 |
| 复数乘法 | 模长相乘，辐角相加 | 振幅调制与相位旋转 | 平面旋转与缩放 | $i$ 旋转 $90^\circ$ |
| 四元数乘法 | 非交换，满足结合律 | 三维旋转的组合 | 旋转顺序不可交换 | 单位四元数表示旋转 |
| 除法 | 乘以逆元（对复数域或四元数域） | 衰减、逆滤波、逆旋转 | 仅当逆存在时成立 | 复数：$z_1/z_2 = z_1\bar{z}_2/\|z_2\|^2$ |
| 实化 | 取实部或标量部，其余置零 | 提取线性分量（梯度/散度方向） | 投影到 $\mathbb{R}$ 轴 | 对实数即自身，对纯虚数得零 |
| 虚化 | 取虚部或向量部，实部置零 | 提取旋转分量（旋度方向） | 投影到纯虚空间 | 对实数得零，对纯虚数得自身 |
| 归一化 | 向量除以其模长 | 提取单位方向 | 模长变为 1 | 用于比较相对方向 |
| 平行 | 两向量成比例（实数系数） | 同向或反向运动 | 夹角 $0^\circ$ 或 $180^\circ$ | 点积 = $\pm$ 模长乘积 |
| 垂直 | 两向量点积为零 | 正交分解、独立分量 | 夹角 $90^\circ$ 或 $270^\circ$ | 一维无垂直，需提升至二维 |
| 角度 | 两向量之间的辐角差或欧几里得夹角 | 相位差、方向差异 | $\cos\theta = \frac{\vec{a}\cdot\vec{b}}{\|\vec{a}\|\|\vec{b}\|}$ | 复数中即辐角差 |
| 距离 | 两向量差的模长 | 空间间隔、势能差 | 满足三角不等式 | 欧几里得距离或曼哈顿距离 |
| 共起点 | 向量起点相同 | 同一参考系、共源场 | 共享原点 | 便于合成与比较 |
| 共终点 | 向量终点相同 | 能量汇聚点、目标一致 | 共享终点 | 表示多种路径到达同一点 |
| 传播 | 向量场随时间的演化 | 波传播、信息传递 | 由梯度、散度、旋度共同决定 | 一阶导数对应速度场 |
| 反射 | 向量关于法线的镜像 | 弹性碰撞、界面反射 | 入射角等于反射角 | 保持模长，改变方向 |
| 折射 | 向量进入不同介质时方向改变 | 光速变化、相位延迟 | 斯涅尔定律 | 界面两侧模长比对应折射率 |
| 法线（一维） | $\operatorname{Normal}(a) := a i$（提升至二维） | 正交方向必须跳出原维度 | 产生纯虚数 | 一维无内在正交方向 |
| 法线（二维） | $\operatorname{Normal}(z) := z i$（旋转 $90^\circ$） | 平面内的正交方向 | 仍在二维内 | 复数乘 $i$ 即旋转 $90^\circ$ |
| 法线（三维） | 通过叉积求得，垂直于两向量张成平面 | 反射与折射界面 | 方向由右手定则确定 | 不唯一（可反向） |
| 极分解 | 复数或四元数分解为模长与单位模的乘积 | 分离强度与旋转信息 | $z = re^{i\theta}$ | 用于独立分析强度与相位 |

### 叉积与法线按维度的层次对应

| 维度 | 数系 | 具有的对称性 | 叉积 | 法线 |
| :--- | :--- | :--- | :--- | :--- |
| 一维 | $\mathbb{Z}, \mathbb{Q}, \mathbb{R}$ | 加法对称（0对称），乘法对称（1对称） | 始终为零 | 跳出至二维 |
| 二维 | $\mathbb{C}$ | 以上 + 平面旋转对称 $U(1)$ | 标量 | 在本空间内 |
| 三维 | $\mathbb{H}$（纯虚部） | 以上 + 三维非交换旋转对称 $SU(2)$ | 向量 | 由叉积给出 |

这些运算与关系构成了场中信息流分析的基本语言。在后续的场章节中，梯度、散度、旋度将作用于向量场上，这些向量运算将为理解场的局域行为提供直接的计算手段。

## 函数

### 定义

设 $X,Y$ 为已构造的数系或其笛卡尔积。函数 $f:X\to Y$ 是一个规则，为每个 $x\in X$ 指派唯一的 $y\in Y$，记作 $f(x)=y$。函数封装了信息场中变量之间的依赖关系。

### 基本初等函数

在实数域上，基本初等函数由以下六类构成，经由有限次四则运算与复合可生成一切初等函数。

| 类别 | 表达式 | 定义域 | 值域 | 关键性质 |
| :--- | :--- | :--- | :--- | :--- |
| 常数函数 | $f(x) = c$ | $\mathbb{R}$ | $\{c\}$ | 不随输入变化 |
| 幂函数 | $f(x) = x^a$ （$a\in\mathbb{R}$） | 视 $a$ 而定 | 视 $a$ 而定 | 当 $a$ 为整数时，即重复乘/除 |
| 指数函数 | $f(x) = a^x$ （$a>0, a\neq 1$） | $\mathbb{R}$ | $\mathbb{R}^+$ | 变化率与自身成正比 |
| 对数函数 | $f(x) = \log_a x$ （$a>0, a\neq 1$） | $\mathbb{R}^+$ | $\mathbb{R}$ | 指数函数的逆 |
| 三角函数 | $\sin x,\; \cos x,\; \tan x,\; \cot x,\; \sec x,\; \csc x$ | 视函数而定 | 周期性有界或间断 | 描述单位圆上的坐标与比值 |
| 反三角函数 | $\arcsin x,\; \arccos x,\; \arctan x$ 等 | $[-1,1]$ 或 $\mathbb{R}$ | 有界区间 | 三角函数的逆 |

这六类基本初等函数构成了表达增长、衰减、振荡等基本演化规律的词汇表。其中指数函数与三角函数在复数域上由欧拉公式 $e^{i\theta}=\cos\theta + i\sin\theta$ 统一，这是周期数学复分析的核心。

### 复变函数

当定义域和值域扩张到 $\mathbb{C}$ 时，便进入复分析领域。周期数学的核心是复指数形式：

$f(\tau) = A(\tau) \cdot e^{i(\omega\tau + \phi)}$

其中 $\tau$ 为实变量（可视为时间或参数），$A(\tau) \in \mathbb{C}$ 为随时间变化的复振幅，$\omega\in\mathbb{R}$ 为角频率，$\phi\in\mathbb{R}$ 为初始相位。$A(\tau)$ 的模长 $|A(\tau)|$ 表示瞬时能量强度，其辐角 $\arg A(\tau)$ 贡献额外的相位调制。当 $A(\tau)$ 为常数时，该函数描绘匀速圆周运动；当 $A(\tau)$ 随时间变化时，它描绘了能量与相位同时演化的复杂周期运动。

任意充分光滑的周期函数均可展开为多个此类函数的级数（傅里叶级数），其中每一项的 $A(\tau)$ 对应于该频率分量的时变频谱包络。

**复指数函数的关键参数**如下表，它们构成描述离散周期序列的基本词汇：

| 参数 | 符号 | 数学定义 | 物理学含义 | 描述 |
| :--- | :--- | :--- | :--- | :--- |
| 周期长度 | $p$ | 函数满足 $f(\tau+p)=f(\tau)$ 的正实数 | 完成一次完整振荡所需的参数跨度 | 决定离散序列的元素总数 |
| 采样率 | $r$ | 单位周期内的采样步长 | 角度分辨率，测量精度 | $r=2\pi/p$ 对应奈奎斯特采样定理 |
| 采样点数 | $n$ | $n = p/r$，一个周期内的采样个数 | 离散状态数 | 周期序列的长度 |
| 频率 | $f$ | $f = 1/p$ | 单位参数的循环次数 | 无量纲频率或时间频率 |
| 复振幅 | $A(\tau)$ | $A(\tau) \in \mathbb{C}$，瞬时复包络 | 能量强度与振幅相位的动态演化 | 模长定能量，辐角定相位偏移 |
| 角频率 | $\omega$ | $\omega = 2\pi f = 2\pi/p$ | 相位变化速率，角速度 | 弧度制下的旋转速度 |
| 相位 | $\phi$ | 初始偏移角，$f(0)=A(0)e^{i\phi}$ | 初始条件，振动起始位置 | 决定序列首元素在周期中的位置 |

### 分贝与能量度量

在周期数学中，能量与振幅的平方成正比：$E(\tau) \propto |A(\tau)|^2$。为在不同状态之间比较能量差异，引入分贝（dB）作为对数标度：

$\text{dB}(\tau) = 20 \log_{10}\left(\frac{|A(\tau)|}{A_0}\right),$

其中 $A_0$ 为参考振幅。分贝值直接反映相对强度：正值表示能量高于参考，负值表示低于参考。在频谱分析与离散周期序列的比较中，分贝提供了紧凑的能量分布表示。

### 高维函数

将定义域扩展至 $\mathbb{R}^n$ 或 $\mathbb{C}^n$，得到多变量函数。对于取值为向量的函数 $\mathbf{F}:\mathbb{R}^n\to\mathbb{R}^n$，可定义散度与旋度（详见场）。四元数函数 $f:\mathbb{H}\to\mathbb{H}$ 则能编码三维空间中的旋转场与旋度演化。

### 极限

**定义** 设函数 $f$ 在点 $a$ 的某去心邻域有定义。若存在实数 $L$，使得对任意 $\varepsilon>0$，存在 $\delta>0$，当 $0<|x-a|<\delta$ 时恒有 $|f(x)-L|<\varepsilon$，则称 $L$ 为 $f(x)$ 当 $x$ 趋近于 $a$ 时的极限，记作

$\lim_{x\to a} f(x) = L.$

单侧极限、无穷极限和无穷远处的极限可类似定义。

**极限的基本性质：**

- 若极限存在，则唯一。
- 四则运算下保持（若各极限存在且分母极限非零）。
- 夹逼定理：若 $g(x)\leq f(x)\leq h(x)$ 且 $\lim g = \lim h = L$，则 $\lim f = L$。

极限概念是连续、导数、积分与级数的共同基础。它使“无限逼近”这一直觉得以形式化，从而精确刻画场中瞬时变化、局部趋向与累积过程。

### 导数

对于实函数 $f:\mathbb{R}\to\mathbb{R}$，导数定义为

$f'(x) := \lim_{h\to 0}\frac{f(x+h)-f(x)}{h}$

若该极限存在。导数度量函数在某点的瞬时变化率。对于复函数，通过实部虚部分别求导，或借助柯西-黎曼条件定义复可导。

高阶导数 $f^{(n)}$ 表示变化率的变化率，揭示了函数行为的更深层结构。对于复指数函数 $f(\tau)=A(\tau)e^{i(\omega\tau+\phi)}$，导数运算涉及 $A(\tau)$ 的变化与旋转：$f'(\tau) = (A'(\tau) + i\omega A(\tau)) e^{i(\omega\tau+\phi)}$。若传播的基本步长为相位旋转 $\Delta\theta$（由场方程的具体形式决定），则从一个状态到下一个状态的推进可由导数操作描述。

| 概念 | 数学含义 | 物理学含义 | 描述 |
| :--- | :--- | :--- | :--- |
| 一阶导数 | $f'(\tau)$，瞬时变化率 | 速度场，局部传播方向 | 当前状态向下一状态的瞬时指向 |
| 二阶导数 | $f''(\tau)$，变化率的变化率 | 加速度场，传播趋势的变化 | 揭示传播是加速还是减速 |
| 三阶导数 | $f'''(\tau)$，加速度的变化率 | 加加速度 | 描述长期趋势的曲率变化 |
| N阶导数 | $f^{(n)}(\tau)$，高阶微分 | 高阶动力学，长程关联 | 通过反复微分追踪远距离的相位偏移 |

### 积分

定积分 $\int_a^b f(x)\,dx$ 定义为黎曼和的极限，表示函数在区间 $[a,b]$ 上的累积效应。积分与导数互为逆运算（微积分基本定理）。在场中，积分用于计算一段演化路径上的总信息流或总相位积累。复积分、路径积分则能捕捉旋转场中的环量——旋度的全局表现。

### 级数展开与积分变换

周期现象的分析强烈依赖于将函数分解为简单成分的叠加。以下工具构成了周期数学的频谱分析方法库，它们将时域信号映射到频域，揭示隐藏的周期结构。

#### 泰勒展开

**定义** 若实（或复）函数 $f$ 在点 $a$ 处无穷可导，则其泰勒级数为

$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x-a)^n.$

当 $a=0$ 时特称为麦克劳林级数。泰勒展开用多项式逼近任意光滑函数，其系数由各阶导数给出。

**物理学含义：** 将局部变化分解为常数项、线性项（梯度）、二次项（曲率）等，是线性化近似、稳定性分析以及非线性效应逐级修正的基础。

#### 傅里叶级数

**定义** 对于周期为 $T$ 的充分光滑函数 $f(t)$，可展开为傅里叶级数：

$f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos\frac{2\pi n t}{T} + b_n \sin\frac{2\pi n t}{T} \right) = \sum_{n=-\infty}^{\infty} c_n e^{i\frac{2\pi n}{T}t}.$

系数 $c_n$ 称为频谱，刻画了信号中各频率分量的振幅与相位。

**物理学含义：** 任何周期运动均可视为一系列简谐振动的叠加。频谱分析将复杂的时域波形转换为清晰的频域能量分布，这是场论中识别主周期、谐波结构以及能量传递路径的核心手段。

#### 基变换（Radix-n FFT）

快速傅里叶变换（FFT）可通过不同基的分解实现，不同基适应不同的序列长度与结构。下表列出常见基及其特性。

| 概念 | 数学含义 | 物理学含义 | 描述 |
| :--- | :--- | :--- | :--- |
| 基-2 FFT | 序列长度 $N=2^k$，逐次二分分解 | 最高效的二分频谱计算 | 要求长度为 2 的幂，最常用 |
| 基-3 FFT | 长度 $N=3^k$，三分递归分解 | 三分支结构的频谱分析 | 适用于三体耦合系统 |
| 基-5 FFT | 长度 $N=5^k$，五分支分解 | 五重对称系统的频谱 | 适合五阶循环结构 |
| 基-7 FFT | 长度 $N=7^k$，七分支分解 | 七重周期结构的高效计算 | 用于七天周期、七声音阶等 |

#### 积分变换工具箱

以下变换提供了从时域到频域（或复频域）的映射，各自适用于不同的信号类型与分析目的。

| 变换 | 数学含义 | 物理学含义 | 描述 |
| :--- | :--- | :--- | :--- |
| DFT (离散傅里叶变换) | 有限长序列的离散频谱 | 数字信号的谐波分解 | 基础变换，FFT是其快速算法 |
| IDFT (逆离散傅里叶变换) | 从频谱重建原序列 | 信号合成 | DFT的逆运算 |

| CTFT (连续时间傅里叶变换) | 连续信号的连续频谱 | 模拟信号的理想频谱 | 理论模型，积分变换 |

### 频谱动力学

在获得了频谱表示之后，场的行为可通过对频谱的操作加以调控与预测。频谱动力学研究信息场中各频率分量的交互、传播与演化，是周期数学分析离散周期序列的核心手段。

**频谱操作：**

| 概念 | 数学含义 | 物理学含义 | 描述 |
| :--- | :--- | :--- | :--- |
| 频谱卷积 | 频域中两个信号的乘积 | 滤波响应，组合效应 | 时域卷积对应频域乘法 |
| 频谱投影 | 信号在某基函数上的内积 | 能量向特定方向/频率的分配 | 提取特定频率分量 |
| 频谱点扰动 | 对单一频率的幅度或相位进行调制 | 谐振激励，局部冲击 | 对应单频外部驱动 |
| 频谱波扰动 | 对一段频带进行调制 | 带宽激励，全局冲击 | 影响一组连续频率 |
| 频谱点扩散 | 用核函数平滑频谱中的某一点 | 信息弥散、热扩散 | 能量向邻近频率均衡化 |
| 频谱波扩散 | 频谱上满足波动方程的演化 | 波传播，周期性扩散 | 频率空间中的波动传递 |

**频谱应用：**

| 概念 | 数学含义 | 物理学含义 | 描述 |
| :--- | :--- | :--- | :--- |
| 频谱分析 | 对场进行傅里叶分解，得到频谱 | 信号分解，识别主周期与谐波 | 找出系统的主导频率 |
| 频谱干涉 | 多个频率分量相干叠加 | 增强或抵消，干涉条纹 | 频率之间的相长或相消 |
| 频谱计算 | 在频域中进行运算预测系统响应 | 系统对输入频谱的输出 | 定量评估演变趋势 |
| 频谱干扰 | 引入噪声或无关频率分量 | 信号污染，负面效应 | 破坏原有频谱结构 |
| 频谱滤波 | 保留或放大特定频段，衰减其他 | 信号提纯，选取有利频段 | 带通、陷波等滤波操作 |
| 频谱耦合 | 不同频率之间发生锁定或同步 | 共振增强，合频效应 | 频率锁定，能量转移 |
| 频谱抑制 | 针对性衰减某些频率分量 | 减弱不利因素，制衡 | 用陷波滤波消除特定频率 |
| 频谱卷积（整体） | 用多个核函数对频谱进行卷积 | 特征提取，综合评估 | 多维度分析场的组合特性 |

## 周期分析要素

在进入场的连续动力学之前，需要建立一套描述周期现象的基本概念。以下要素将连续的时间演化、离散的采样、频域的表示以及物理量的测量统一在一个框架内。它们构成了从函数分析到场论的桥梁——场在每一时刻的状态可经由采样和频谱分析被捕捉为离散的周期序列元素。

| 概念 | 数学含义 | 物理学含义 | 描述 |
| :--- | :--- | :--- | :--- |
| 时间 | 演化参数，通常取为实变量 $t$ | 因果序列的推进方向 | 一维连续参量，标记场的演化帧序 |
| 周期 | 函数满足 $f(t+T)=f(t)$ 的最小正数 $T$ | 完成一次完整循环所需的时长 | 由场的边界条件与本征频率决定 |
| 频率 | $f=1/T$，周期的倒数 | 单位时间内循环的次数 | 与角频率 $\omega=2\pi f$ 对应 |
| 分辨率 | 时域采样间隔 $\Delta t$ 或频域间隔 $\Delta f$ | 测量精度与区分能力 | 采样越密，可分辨的频率成分越丰富 |
| 趋势 | 信号的低频包络或梯度方向 | 系统演化的宏观指向 | 去除高频波动后的长程走向，势能梯度的反映 |
| 相量 | 复振幅 $Ae^{i\phi}$，忽略时间因子 $e^{i\omega t}$ | 稳态振动的幅度与相位表示 | 将微分方程化为代数方程，用于分析稳态响应 |
| 波形 | 时域信号 $f(t)$ 的具体形态 | 场振动的瞬时值随时间的轨迹 | 可以是正弦、方波、锯齿波等任意形态 |
| 频谱 | 傅里叶系数 $c_n$ 或傅里叶变换 $\hat{f}(\omega)$ | 各频率分量的能量与相位分布 | 揭示周期性背后的谐波结构 |
| 样本 | 在离散时刻 $t_k$ 采集的场值 $f(t_k)$ | 对连续场的离散快照 | 单次测量或单点记录 |
| 采样器 | 采样函数或采样序列 $\{t_k\}$ | 决定何时以何种间隔进行测量的规则 | 采样频率需满足奈奎斯特定理以避免混叠 |

这些要素将函数论中的连续描述（时间、波形、频谱）与离散化操作（采样、分辨率）统一起来。在根本场的视角下，每一次采样都是场在特定相位下的快照，而采样序列的总和即构成一个离散周期序列——这正是周期数学所要分类与预测的核心对象。

## 场

向量函数的微分与积分运算，构成了描述连续介质局部行为的语言。在周期数学中，这些算子集中体现于一个根本的场，其动力学统一包含梯度、散度、旋度。

### 定义

周期数学假设存在一个根本的场，其动力学由统一包含梯度、散度、旋度的演化方程描述。在这种动力学下，该场能自发涌现出一系列稳定的、孤立的局部结构，而这些结构的完整分类，恰好对应于人类已建立的数系（$\mathbb{N}$ 至 $\mathbb{H}$）及其上的代数对称性。

在数学描述中，我们通常将这一根本场投影为以下两类基本形式：

- **标量场** $\phi: \mathcal{M}\to \mathbb{R}$：每点赋予一个信息密度或势值。
- **向量场** $\mathbf{F}: \mathcal{M}\to T\mathcal{M}$：每点赋予一个信息流的大小和方向。

场的演化动力学是周期数学的核心研究对象。其局部行为完全由三类基本算子刻画：梯度、散度、旋度。三者分别控制信息密度的定向差异、信息流的源汇强度以及信息流的闭合循环。

### 梯度

**定义** 对标量场 $\phi$，其梯度 $\nabla\phi$ 是一个向量场，指向 $\phi$ 增加最快的方向，大小等于该方向的方向导数：

$\nabla\phi = \left( \frac{\partial\phi}{\partial x_1}, \frac{\partial\phi}{\partial x_2}, \dots, \frac{\partial\phi}{\partial x_n} \right)$

**场的意义** 梯度揭示了信息密度的空间不对称性。场倾向于从低密度区流向高密度区（或反之，取决于场的约定）。梯度方向定义了局部“增加”的指向，是所有加法对称（正/负方向）的场论根源。当场在梯度作用下形成周期势阱时，便自然产生离散的稳定态序列——这正是整数“序列方向”的动力学源头。

### 散度

**定义** 对向量场 $\mathbf{F}=(F_1,\dots,F_n)$，其散度是一个标量场：

$\nabla\cdot\mathbf{F} = \frac{\partial F_1}{\partial x_1} + \frac{\partial F_2}{\partial x_2} + \cdots + \frac{\partial F_n}{\partial x_n}$

**场的意义** 散度度量了信息流在某点的“源”或“汇”的强度。正散度表示信息在此处生成（向外发散），负散度表示信息在此处湮灭（向内汇聚）。散度的对比产生了缩放关系——两处散度强度的比值定义了放大或缩小，这是有理数乘法对称（1对称）的场论根源。当场出现周期性的源汇结构时，便形成可量化的比例关系。

### 旋度

**定义** 对三维向量场 $\mathbf{F}$，其旋度是一个向量场：

$\nabla\times\mathbf{F} = \left( \frac{\partial F_3}{\partial x_2} - \frac{\partial F_2}{\partial x_3},\; \frac{\partial F_1}{\partial x_3} - \frac{\partial F_3}{\partial x_1},\; \frac{\partial F_2}{\partial x_1} - \frac{\partial F_1}{\partial x_2} \right)$

在二维情形，旋度退化为标量 $\frac{\partial F_2}{\partial x_1} - \frac{\partial F_1}{\partial x_2}$。

**场的意义** 旋度度量了信息流的局部旋转强度。旋度不为零的区域，信息流并不直来直往，而是绕局部轴形成闭合涡旋。旋度是旋转对称（单位圆对称与三维旋转对称）的场论根源：二维旋度诱导平面旋转，对应复数相位；三维旋度的三个独立分量对应四元数虚部的非交换代数。当场出现稳定的旋度结构时，便形成闭合周期——圆周运动、涡旋、循环序列。

### 三算子的协同与周期涌现

梯度、散度、旋度并非独立作用。场的演化方程通常同时包含这三类项。例如，一类简化模型可写为：

$\frac{\partial\mathbf{F}}{\partial t} = \alpha\nabla\phi + \beta\nabla\cdot\mathbf{F} + \gamma\nabla\times\mathbf{F}$

其中 $\alpha,\beta,\gamma$ 为耦合系数。在这样的动力学下，场自然涌现出分立的稳定结构：周期势阱（自然数的离散节点）、定向流（整数位移）、聚散比例（有理数缩放）、连续强度分布（实数场势）、闭合涡旋（复数相位）、三维旋转涡管（四元数空间方向）。

这些稳定结构的采样与组合，形成了具有固定相位差的离散周期序列。序列中的每个元素都是场在某特定相位下的状态快照，相邻元素之间由固定的算子操作关联——有的对应梯度推进，有的对应散度缩放，有的对应旋度相位旋转。整个序列构成一个完整的周期循环，其长度和结构由场的本征频率和边界条件决定。

因此，根本场的动力学不仅生成了人类认知中所有的数系与对称性，更将它们组织为一个个可观测的周期序列。周期数学的任务，正是用前述所有数学工具——从自然数到四元数，从向量运算到复变函数，从极限导数积分到梯度散度旋度——去分类、分析、预测这些周期序列的构成与演化规律。它不借助任何直观隐喻或文化符号，仅凭场的算子动力学，就重现了“道生一，一生二，二生三，三生万物”的逻辑：从无结构的基底，到离散的计数，到有方向的序列，再到比例、连续、旋转，最终生成万物繁复的周期交响。

## 波与场：动态关系与几何表示

在完成了数系、向量、函数与场的构建之后，我们来讨论波与场之间的动态关系，以及它们在几何上的统一表示。

### 波形与频谱的几何本质

波形是连续的复变函数 $f(\tau)$，或离散采样点的积分重构。其内部的值可以是任意有方向的数——实数、复数或四元数——天然支持旋转（相位推进）与缩放（振幅变化）。波形是信息在时域中的直接呈现。

频谱是波形经傅里叶变换后在频域中的离散点集合。频域的分辨率由采样率决定。每个频点的值同样是带方向的数，承载该频率分量的振幅（模长）与相位（辐角）。频谱是隐藏在波形背后的周期结构。

### 场的产生：磁化过程

场的产生是一个"磁化"过程——在无序状态下，各点的信息流方向随机，互相抵消，宏观上无结构。当外部条件或内部耦合使局部信息流趋向有序排列时，散度（源汇）、梯度（势差）、旋度（涡旋）开始显现，场便诞生了。场负责维护时间与空间的静态关系——势阱在哪里，源在何处，涡旋绕何轴旋转。

### 场、波、频谱的层级关系

场是最根本的结构，它定义了空间每一点的算子状态（梯度、散度、旋度的分布）。当场与场之间发生干扰或耦合时，静态的平衡被打破，扰动以波的形式在空间中传播。波在传播过程中与局部的梯度、散度、旋度不断相互作用——梯度加速或减速传播，散度汇聚或发散能量，旋度赋予波以旋转偏振。

波在时域中的形态（波形）经采样和傅里叶变换后，得到频谱。频谱揭示了波中蕴藏的所有周期成分——哪些频率主导，哪些被抑制，各频率之间的相位关系如何。

三者的关系可概括为：

**场 → 干扰/耦合 → 波 → 采样+变换 → 频谱**

反向过程同样成立：从频谱可合成波，波的能量分布可反过来改变场的源汇与涡旋结构。

### 操作的分层

不同类型的操作可作用于不同层级：

**作用于点（局部）：**

- **点干扰**：对场、波或频谱中的单一点施加脉冲激励，改变其瞬时值。
- **点扩散**：以某点为中心，用扩散核将能量平滑到邻近区域（时域平滑或频域平滑）。

**作用于波（时域）：**

- **波干扰**：两列或多列波在空间中相干叠加，产生干涉条纹。
- **波耦合**：不同频率或不同方向的波之间发生能量交换，如频率锁定、模式转换。

**作用于场（空间）：**

- **场干扰**：两个场的梯度、散度、旋度分布发生重叠，改变彼此的源汇与涡旋位置。
- **场耦合**：场的演化方程中的非线性项导致不同场分量之间的能量传递与结构重组。

**作用于频谱（频域）：**

- **频谱干扰**：多个频率分量的相干叠加，增强或抵消特定频率。
- **频谱耦合**：不同频率之间锁定相位关系，形成谐波结构或频率梳。

### 统一几何图像

所有操作都可以在相空间中获得几何表示。波形是相空间中的一条轨迹（参数曲线），场是相空间中的向量场（每点一个箭头），频谱是复平面上的一组点集（每个频率分量一个点）。

- **旋转**对应相位的推进——在复平面或四元数空间中绕原点或单位球转动。
- **缩放**对应振幅的增强或衰减——向径的伸缩，能量的对数变化。
- **扩散**对应点集在相空间中的平滑化——点向邻近点"泄露"能量。
- **干扰**对应向量的加减——相空间中两个向量的合成或抵消。
- **耦合**对应非线性变换——两个频率分量相乘产生和频与差频，在频谱中表现为新点位的出现。

### 物体与能量的波本质

在周期数学中，物体与能量均以波的形式在空间中传递。一个"粒子"不过是波包——在空间局域化的波形，其内部承载着旋度（自旋）、梯度（动量）和散度（质量/能量源汇）。波包在传播过程中，不断与空间中的梯度（势场）、散度（源汇场）、旋度（涡旋场）进行相互作用——这正是经典力学与量子力学中各种力的统一描述。引力对应散度场（质量汇聚），电磁力对应旋度场（电荷旋转），强弱相互作用对应梯度场（核内势阱）。

这一视角将场的三类算子（梯度、散度、旋度）与波的行为（传播、干涉、耦合）以及频谱的结构（谐波、噪声、共振峰）编织为一个自洽的整体——宇宙中的一切现象，都是信息场中波与算子的几何舞蹈。
