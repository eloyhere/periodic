## 基础公设

以下公设不属于数学推导，而是为周期数学提供哲学起点与研究动机。

**公设 1（信息本体）**
宇宙的基本构成是信息及其关系，物质与能量是信息关系的衍生现象。

**公设 2（数学是真子集）**
数学是宇宙信息关系中可被形式化、可计算的那一部分。并非所有信息关系都已被数学化。

**公设 3（周期普遍性）**
周期是信息演化的普遍特征。在适当的条件下，信息关系自然涌现出稳定的周期结构。

> 数学是宇宙信息关系中可被形式化、可计算的那一部分。
> 数学部分从下一节正式开始，此后不再引用上述哲学公设。

## 方法论

周期数学接受现代数学的全部标准构造，不重新定义已有概念。其工作是：

1. **揭示对称性与方向性**作为周期性的基础；
2. **给出周期性在群论语言中的严格定义**，从对称性出发；
3. **揭示数系扩张与对称群增加之间的结构对应**；
4. **为每一种数系建立对应的周期函数**；
5. **将傅里叶分析及其他积分变换纳入周期群上的调和分析框架**；
6. **以范畴化操作统一描述波与频谱的计算**。

## 第一部分：对称性与方向性——周期性的基础

周期性的根源是对称性。一个系统呈现出周期行为，意味着存在某种变换，使得系统在该变换下不变。这种变换的集合构成群，而群作用的方向性赋予周期以“指向”。本章阐述对称性如何衍生出方向性，以及方向性如何成为周期性的数学基础。

### 1.1 对称性

**定义 1（对称性）**
设 $S$ 为一个集合。$S$ 上的一个对称性是一个保持某种结构不变的可逆变换 $\sigma: S \to S$。$S$ 上所有对称性的集合在变换的复合下构成群，称为 $S$ 的对称群（或自同构群）。

**例 1（几何对称）**
*   单位圆 $S^1$ 的对称群是正交群 $O(2)$（所有旋转与反射）。
*   正 $n$ 边形的对称群是二面体群 $D_n$。
*   全空间 $\mathbb{R}^n$ 在平移下的对称群是加法群 $\mathbb{R}^n$。

**例 2（代数对称）**
*   整数加法群 $(\mathbb{Z},+)$ 具有对称性 $x \mapsto -x$。这个对称性生成了 $C_2$ 群（二阶循环群）。
*   非零有理数乘法群 $(\mathbb{Q}^\times,\times)$ 具有对称性 $x \mapsto x^{-1}$。
*   复数域 $\mathbb{C}$ 具有旋转对称性 $z \mapsto e^{i\theta}z$，构成 $U(1)$ 群。

### 1.2 方向性

**定义 2（方向性）**
设 $G$ 为作用于光滑流形 $M$ 上的李群，$\mathfrak{g}$ 为其李代数。对于生成元 $X \in \mathfrak{g}$，其在 $p \in M$ 处诱导的向量场 $X_M(p)$ 给出了 $G$ 在 $p$ 处的无穷小作用方向。若 $G$ 为离散群，方向由生成元在状态空间中的走向给出。若 $G$ 有多组生成元，则存在多个独立的方向维度。

方向不是独立于对称性而存在的，它是对称群作用的产物。同一个群的不同生成元可以产生不同的方向。

### 1.3 三大基本对称与方向

在数系的扩张中，三种最根本的对称操作反复出现，它们各自对应一种基本的方向类型。

**（1）加法对称：0 对称与位移方向**
在整数 $\mathbb{Z}$ 中，对称操作 $x \mapsto -x$ 以 $0$ 为不动点。这一对称将整数分为正、负两个半轴。正数表示沿一个方向的累积，负数表示沿相反方向的累积。由此产生的方向称为**位移方向**。
*   **性质**：加法对称是对合——应用两次回到自身：$-(-x) = x$。生成 $C_2$ 群。

**（2）乘法对称：1 对称与缩放方向**
在非零有理数 $\mathbb{Q}^\times$ 中，对称操作 $x \mapsto x^{-1}$ 以 $1$ 为不动点。这一对称将正有理数分为放大（$x > 1$）和缩小（$0 < x < 1$）两个区域。由此产生的方向称为**缩放方向**。
*   **性质**：乘法对称也是对合：$(x^{-1})^{-1} = x$。但缩放方向是连续的——存在不同程度的放大与缩小，而不仅仅是两个离散方向。

**（3）旋转对称：$i$ 对称与相位方向**
在复数 $\mathbb{C}$ 中，对称操作 $z \mapsto iz$（乘以虚数单位 $i$）旋转 $90^\circ$。$i$ 的幂次生成离散旋转群 $\{1, i, -1, -i\}$，对应四个基本方向。连续的旋转构成 $U(1)$ 群 $z \mapsto e^{i\theta}z$。由此产生的方向称为**相位方向**或**旋转方向**。
*   **性质**：旋转对称不是对合（$i^2 = -1 \neq 1$），需要四次操作回到自身（$i^4 = 1$）。在三维中，旋转的非交换性由 $SU(2)$（单位四元数群）描述。

### 1.4 从对称性到周期性

对称性是比周期性更原始的概念。周期性是对称性在某个特定方向上的反复作用所呈现的结构。

**命题 1（对称性生成闭合周期与轨道）**
设 $G$ 为对称群，$H = \langle g \rangle \subset G$ 是由某个对称操作 $g \in G$ 生成的循环子群。
*   若 $g$ 的阶有限（即存在最小正整数 $n$ 使得 $g^n = e$），则 $g$ 的反复作用形成一个闭合周期，周期长度为 $n$。
*   若 $g$ 的阶无限（即对所有 $k \neq 0$ 有 $g^k \neq e$），则 $g$ 的反复作用形成一个无限轨道，永不返回起点。

**论证**：若 $g^n = e$，则作用序列 $s, g\cdot s, g^2\cdot s, \dots$ 在 $n$ 步后首次返回 $s$，形成一个长为 $n$ 的循环。若 $g$ 的阶无限，则序列中无重复元素，形成无限轨道。$\square$

**推论 1**：三种基本对称性对应三种基本的周期或轨道类型：
*   加法对称（$C_2$）→ 二态周期（正/负交替）
*   乘法对称（对合 $x \mapsto x^{-1}$）→ 等比轨道（放大与缩小交替）
*   旋转对称（$U(1)$）→ 圆周周期（相位循环）

### 1.5 对称性中心：0、1、i

在三大基本对称中，有三个特殊的数充当对称性的“中心”或“生成元”。
*   **0** 是加法对称 $x \mapsto -x$ 的不动点。它是正与负的分界，是位移方向的基准。
*   **1** 是乘法对称 $x \mapsto x^{-1}$ 的不动点。它是放大与缩小的分界，是缩放方向的基准。
*   **i** 是旋转群 $U(1)$ 的生成元。乘以 $i$ 就是旋转 $90^\circ$。$i$ 的幂次生成整个离散旋转群。

> 此三个特殊数将在场论中与梯度、散度、旋度建立解释性关联（见第七部分）。此关联属于理论解释，非已有数学定理。

## 第二部分：数系

数是信息关系中稳定结构的符号化。数系是按对称性与完备性逐级扩展的代数系统。每一次数系扩张都引入新的对称性，从而引入新的方向性。

### 2.1 自然数 $\mathbb{N}$

**公理（皮亚诺算术）**
1. $0$ 是自然数。
2. 每个自然数 $n$ 有唯一后继 $S(n)$，且 $S(n)$ 是自然数。
3. $0$ 不是任何自然数的后继。
4. 若 $S(n) = S(m)$，则 $n = m$。
5. 归纳公理：若性质 $P$ 满足 $P(0)$，且 $P(n)\Rightarrow P(S(n))$ 对所有 $n$ 成立，则 $P$ 对所有自然数成立。

**约定**：$1:=S(0),\;2:=S(1),\;\ldots$

**运算（递归定义）**
*   加法：$n+0:=n,\; n+S(m):=S(n+m)$
*   乘法：$n\times 0:=0,\; n\times S(m):=(n\times m)+n$
*   序：$n\leq m\iff \exists k\in\mathbb{N},\; n+k=m$

$(\mathbb{N},+,\times,\leq)$ 构成有序交换半环。自然数仅有“大小”，无逆元，无对称性。它是纯粹的标量，回答“有多少个离散元”。

### 2.2 整数 $\mathbb{Z}$

**构造** 在 $\mathbb{N}\times\mathbb{N}$ 上定义等价关系 $\sim$：
$(a,b)\sim(c,d) \iff a+d = b+c$

整数集 $\mathbb{Z}:=(\mathbb{N}\times\mathbb{N})/\sim$，等价类 $[(a,b)]$ 表示位移“从 $b$ 到 $a$”。

**运算**
*   加法：$[(a,b)]+[(c,d)]:=[(a+c,\;b+d)]$
*   零元：$0_{\mathbb{Z}}:=[(0,0)]$
*   加法逆：$-[(a,b)]:=[(b,a)]$
*   乘法：$[(a,b)]\times[(c,d)]:=[(ac+bd,\;ad+bc)]$
*   单位元：$1_{\mathbb{Z}}:=[(1,0)]$
*   序：$[(a,b)]\leq[(c,d)]\iff a+d\leq b+c$

$(\mathbb{Z},+,\times,\leq)$ 构成有序交换环。每个元素都有加法逆元，产生关于 0 的加法对称。周期数学将此解释为位移方向的代数来源。整数可视为一维格点上的 $\mathbb{Z}$-模向量。

**自然数嵌入**：$n\mapsto[(n,0)]$ 保持加乘结构。

### 2.3 有理数 $\mathbb{Q}$

**构造** 令 $\mathbb{Z}^\times:=\mathbb{Z}\setminus\{0\}$。在 $\mathbb{Z}\times\mathbb{Z}^\times$ 上定义 $\approx$：
$(a,b)\approx(c,d) \iff a\times d = b\times c$

有理数集 $\mathbb{Q}:=(\mathbb{Z}\times\mathbb{Z}^\times)/\approx$，等价类记为 $\frac{a}{b}$。

**运算**
*   加法：$\frac{a}{b}+\frac{c}{d}:=\frac{a\times d+b\times c}{b\times d}$
*   零元：$\frac{0}{1}$，加法逆：$-\frac{a}{b}:=\frac{-a}{b}$
*   乘法：$\frac{a}{b}\times\frac{c}{d}:=\frac{a\times c}{b\times d}$
*   单位元：$\frac{1}{1}$，乘法逆：若 $a\neq 0$，$(\frac{a}{b})^{-1}:=\frac{b}{a}$
*   序：$\frac{a}{b}>0\iff a\times b>0$，并由此推导全序。

$(\mathbb{Q},+,\times,\leq)$ 构成有序域。非零元素不仅有加法逆（0 对称），还具有乘法逆，产生关于 1 的乘法对称。周期数学将此解释为缩放方向的代数来源——$x$ 与 $x^{-1}$ 关于 1 对称，分别表示放大与缩小。

**整数嵌入**：$n\mapsto\frac{n}{1}$。

### 2.4 实数 $\mathbb{R}$

**构造（戴德金分割）** 实数集 $\mathbb{R}$ 由所有满足以下条件的下组 $L\subset\mathbb{Q}$ 构成：
1. $L\neq\emptyset$，$L\neq\mathbb{Q}$。
2. $L$ 没有最大元素。
3. 若 $x\in L$，$y\in\mathbb{Q}$ 且 $y<x$，则 $y\in L$（向下封闭）。

有理数 $q$ 对应下组 $\{x\in\mathbb{Q}\mid x<q\}$。

**运算**
*   序：$r\leq s\iff L_r\subseteq L_s$
*   加法：$L_{r+s}:=\{x+y\mid x\in L_r,\;y\in L_s\}$
*   乘法：先对正实数定义为 $\{xy\mid x\in L_r,y\in L_s,x,y>0\}\cup\mathbb{Q}_{\leq 0}$，再依符号扩展。

$\mathbb{R}$ 满足最小上界性质，即完备性。实数保留了加法逆和乘法逆，并引入连续性。

### 2.5 复数 $\mathbb{C}$

**构造** 复数集 $\mathbb{C}:=\mathbb{R}^2$。定义加法为逐分量相加，乘法为
$(a,b)\times(c,d):=(ac-bd,\;ad+bc)$

记 $a+bi:=(a,b)$，$i:=(0,1)$，则 $i^2=-1$。

**代数结构** $\mathbb{C}$ 是二维交换 $\mathbb{R}$-代数，且为域。每个非零复数有乘法逆 $(a+bi)^{-1}=\frac{a-bi}{a^2+b^2}$。

**极形式与旋转对称**
任意 $z\in\mathbb{C}$ 可写为 $z=re^{i\theta}$，其中 $r=|z|=\sqrt{a^2+b^2}$ 为模长，$\theta$ 为辐角。乘法 $z\mapsto e^{i\phi}z$ 保持模长不变，使辐角增加 $\phi$，即平面旋转。单位复数群 $\{e^{i\theta}\}$ 同构于 $U(1)$。周期数学将此解释为相位旋转方向的代数载体。

### 2.6 四元数 $\mathbb{H}$

**构造** 四元数集 $\mathbb{H}:=\mathbb{R}^4$，元素记作 $q=a+bi+cj+dk$。加法逐分量进行。乘法由以下规则双线性扩张：
*   $i^2=j^2=k^2=ijk=-1$
*   $ij=k,\; ji=-k$; $jk=i,\; kj=-i$; $ki=j,\; ik=-j$

乘法非交换，但满足结合律。$\mathbb{H}$ 是非交换 $\mathbb{R}$-除法代数。

**共轭与范数** $\bar{q}:=a-bi-cj-dk$，$|q|:=\sqrt{a^2+b^2+c^2+d^2}$。非零元的逆 $q^{-1}=\bar{q}/|q|^2$。

**旋转对称** 单位四元数群 $S^3$ 同构于 $SU(2)$，通过 $v\mapsto qvq^{-1}$（$v$ 为纯虚四元数）实现三维空间旋转。不同轴旋转的非交换性对应四元数乘法的非交换性。

### 2.7 数系总览与对称性中心

| 数系 | 新增对称性 | 周期数学解释 |
| :--- | :--- | :--- |
| $\mathbb{N}$ | 无 | 纯大小 |
| $\mathbb{Z}$ | 加法对称 $x\mapsto -x$（0 对称） | 位移方向 |
| $\mathbb{Q}$ | 乘法对称 $x\mapsto x^{-1}$（1 对称） | 缩放方向 |
| $\mathbb{R}$ | 完备化 | 连续强度 |
| $\mathbb{C}$ | $U(1)$ 旋转对称 | 相位旋转方向 |
| $\mathbb{H}$ | $SU(2)$ 非交换旋转对称 | 空间旋转方向 |

在数系的逐级构造中，有三个特殊的数反复出现。**0** 是加法单位元与加法逆的对称中心——正负方向的分界。**1** 是乘法单位元与乘法逆的对称中心——放大与缩小的分界。**i** 是旋转群 $U(1)$ 的生成元——乘以 $i$ 即旋转 $90^\circ$。

## 第三部分：向量

向量是标量域（或系数环）上的模或向量空间中的元素。每一种数系作为标量域，自然产生对应的方向结构。

### 3.1 各类数向量的性质

| 向量类型 | 分量域 | 模长定义 | 方向表示 | 物理对应 |
| :--- | :--- | :--- | :--- | :--- |
| 自然数向量 | $\mathbb{N}^n$ | 坐标和 | 序列前进方向 | 步数计数 |
| 整数向量 | $\mathbb{Z}^n$ | 曼哈顿距离或格点范数 | 各分量符号与比值 | 晶格位移、离散位移 |
| 有理数向量 | $\mathbb{Q}^n$ | 欧几里得范数 | 各分量比值 | 比例缩放后的位置 |
| 实数向量 | $\mathbb{R}^n$ | $\sqrt{\sum x_i^2}$ | 与坐标轴的夹角 | 速度、力、场强 |
| 复数向量 | $\mathbb{C}^n$ | $\sqrt{\sum \|z_i\|^2}$ | 各分量的辐角与相对相位 | 电磁波振幅与相位、量子态 |
| 四元数向量 | $\mathbb{H}^n$ | $\sqrt{\sum \|q_i\|^2}$ | 各分量的旋转轴与角度 | 三维旋转、刚体姿态 |

### 3.2 向量关系与运算

| 概念 | 数学含义 | 方向性关联 | 特性 |
| :--- | :--- | :--- | :--- |
| **加法** | 向量逐分量相加 | 位移方向的合成 | 交换律、结合律成立 |
| **减法** | 向量逐分量相减 | 位移方向的分解 | 加上逆向量 |
| **标量乘法** | 各分量乘以同一标量 | 缩放方向（由 1 对称控制） | 改变模长，保持或反转方向 |
| **点积（内积）** | $\vec{a}\cdot\vec{b} = \sum a_i b_i$ | 方向投影的大小 | 交换律，结果为数；垂直时为零 |
| **投影** | $\operatorname{proj}_{\vec{b}}\vec{a} = \frac{\vec{a}\cdot\vec{b}}{\vec{b}\cdot\vec{b}}\vec{b}$ | 方向分解 | 结果与 $\vec{b}$ 共线 |
| **叉积** | $0$ / 标量 $ad-bc$ / 向量 | 旋转方向的度量 | 反交换 |
| **复数/四元数乘法** | 旋转与缩放 | 相位/空间旋转方向的复合 | 非交换（四元数） |
| **实化/虚化** | 提取线性/旋转分量 | 分离位移方向与旋转方向 | 投影操作 |
| **归一化** | 向量除以其模长 | 提取单位方向 | 模长变为 1 |
| **平行/垂直** | 成比例 / 点积为零 | 位移方向一致或正交 | 夹角 $0^\circ$ 或 $90^\circ$ |
| **角度/距离** | 辐角差或欧几里得夹角 / 差的模长 | 方向差异 / 位移幅度的差异 | 满足三角不等式 |
| **传播** | 向量场随时间的演化 | 方向随时间的演变 | 由梯度、散度、旋度决定 |
| **反射/折射** | 关于法线镜像 / 方向偏转 | 位移方向反转/偏转 | 入射角等于反射角/斯涅尔定律 |

## 第四部分：周期体系

周期数学的核心创新之一，是将“周期”从函数的属性提升为状态集合上的群作用结构。以下定义使用群论与表示论的标准语言。

### 4.1 周期关系的群论确定

**定义 3（$G$-周期性）**
设 $S$ 为集合，$G$ 为作用于 $S$ 的变换群。若存在子集 $F\subset S$（基本域），使得 $S$ 中每个元素均可通过 $G$ 中某个变换作用于 $F$ 中某个元素得到，且不同轨道互不相交：
$$S = \bigsqcup_{f\in F} G\cdot f,$$
则称 $S$ 关于 $F$ 具有 $G$- periodicity。$F$ 中的元素称为对称点，$G$ 称为该系统的周期群。周期结构是群作用的不变量，而周期函数只是该结构上的一种表示。

**定义 4（周期函数——表示论形式）**
设 $S$ 具有 $G$-周期性，$V$ 为向量空间，$\rho: G\to\text{GL}(V)$ 为表示。若 $\psi: S\to V$ 满足
$$\psi(g\cdot s) = \rho(g)\psi(s),$$
则称 $\psi$ 为关于 $G$ 的周期函数。$\rho=\text{id}$ 对应通常的周期函数；$\rho\neq\text{id}$ 对应协变周期函数（如 Bloch 波）。

**定义 5（余数作为周期函数）**
设 $m \in \mathbb{N}^+$。商集 $\mathbb{Z}/m\mathbb{Z}$ 构成 $m$ 阶循环群 $\mathbb{Z}_m$。映射
$$r_m: \mathbb{Z} \to \mathbb{Z}_m, \quad r_m(n) = n \bmod m$$
满足 $r_m(n+m) = r_m(n)$，是整数平移群作用下的离散周期函数。余数将无限的整数序列折叠为有限循环群上的轨道。

### 4.2 周期度量参数

| 参数 | 符号 | 数学定义 | 物理含义 |
| :--- | :--- | :--- | :--- |
| **周期长度** | $p$ | $f(\tau+p)=f(\tau)$ 的最小正实数，或生成元步长 | 完成一次完整循环的参数跨度 |
| **频率** | $f$ | $f = 1/p$ | 单位参数内的循环次数 |
| **角频率** | $\omega$ | $\omega = 2\pi f = 2\pi/p$ | 相位变化的角速度 |
| **采样点数** | $n$ | 单个周期内选取的离散观测点个数 | 离散化程度 |
| **采样间隔** | $\Delta t$ | $\Delta t = p/n$ | 相邻采样点间的参数跨度 |
| **采样率** | $f_s$ | $f_s = 1/\Delta t = n/p$ | 单位参数内的采样次数 |
| **相位** | $\phi$ | 相对基本域的偏移量，$\phi \in [0, 2\pi)$ 或 $[0, p)$ | 振动起始位置 |
| **振幅** | $A$ | $A = \sup_{\text{一个周期}} \|\psi - \bar{\psi}\|$ | 能量强度 |

### 4.3 周期与轨道的分类

**定义 6（静态周期）**
若周期群 $G$ 为循环群 $G = \langle g \rangle$（或循环群的直积），且生成元 $g$ 的作用在度量上为等距平移——即存在常数 $p > 0$，使得对任意 $s \in S$，有 $d(s, g\cdot s) = p$——则称 $S$ 具有静态周期（固定周期），$p$ 为最小周期。余数函数 $n \mapsto n \bmod m$ 是静态周期的离散原型。

**定义 7（动态周期）**
若生成元作用距离不是常数，即存在局部周期函数 $p: S \to \mathbb{R}_{>0}$，使得 $d(s, g\cdot s) = p(s)$ 且 $p(s)$ 不是常数函数，则称 $S$ 具有动态周期。此定义适用于可局部周期化的系统。李群表述：存在李代数元素 $X$ 和光滑标度函数 $\lambda: S \to \mathbb{R}^+$，使得无穷小作用为 $s \mapsto \exp(\lambda(s)X) \cdot s$。

**定义 8（级数周期）**
若局部周期函数 $p(s)$ 可展开为傅里叶级数 $p(s) = p_0 + \sum (a_n\cos\frac{2\pi ns}{P} + b_n\sin\frac{2\pi ns}{P})$，则称系统具有级数周期。若所有 $a_n = b_n = 0$，退化为静态周期。

**定义 9（无限轨道）**
若变换群 $G$ 为非紧群（如 $\mathbb{Z}$ 或 $\mathbb{R}$），轨道沿一个方向无限延伸而不返回起点，则称系统具有无限轨道。自然数序列、双曲线轨道均属此类。注：在传统术语中，“周期”通常指闭合轨道。周期数学将闭合周期与无限轨道统一纳入广义的周期结构体系。

**定义 10（吸引轨道）**
若轨道趋近于一个极限点 $s_*$ 且趋近具有自相似结构——即存在 $0 < \lambda < 1$ 使得 $d(g^{k+1}\cdot s, s_*)/d(g^k \cdot s, s_*) \to \lambda$——则称系统具有吸引轨道。等比数列 $\{r^n\}$（$0 < r < 1$）是最基本的实例。

### 4.4 周期组合与共振

**定义 11（周期组合）**
若系统由两个独立周期过程组成（周期群 $G_1, G_2$，阶数 $n_1, n_2$），耦合后整体周期群为 $G_1 \times G_2$ 的商群，阶数为 $\operatorname{lcm}(n_1, n_2)$ 的因子。

若 $n_1/n_2 = m/k$（既约分数），则称 $m:k$ 为共振比，$|m-k|$ 为共振阶。低阶共振更稳定。天文学实例：木星伽利略卫星的 $1:2:4$ 周期比。

## 第五部分：每种数系的一般周期函数

周期数学为每一种数系建立一般的周期函数，揭示数系结构与其上周期行为之间的内在关联。

### 5.1 自然数上的周期函数

**定义 12（自然数上的周期序列）**
$f: \mathbb{N} \to V$ 是终归周期序列，若存在 $p$ 使 $f(n+p)=f(n)$ 对所有充分大的 $n$ 成立。若对所有 $n$ 成立，则为纯周期序列。自然数上的周期是“单向”的，是无限轨道在离散情形下的表现。

### 5.2 整数上的周期函数

**定义 13（整数上的周期函数）**
$f: \mathbb{Z} \to V$ 是周期为 $p$ 的周期函数，若 $f(n+p)=f(n)$ 对所有 $n\in\mathbb{Z}$ 成立。一般形式：$f(n)=g(n \bmod p)$，其中 $g:\mathbb{Z}_p \to V$。余数函数是最基本的例子。

### 5.3 有理数上的周期函数

**定义 14（有理数上的加法周期函数）**
$f: \mathbb{Q} \to V$ 满足 $f(x+q)=f(x)$，周期群为 $q\mathbb{Z}$。

**定义 15（有理数上的乘法周期函数）**
$f: \mathbb{Q}^+ \to V$ 满足 $f(rx)=f(x)$，$r\neq 1$，周期群为乘法群 $\langle r \rangle$。等比序列 $f(k)=r^k$ 是乘法群上轨道的基本实例。

### 5.4 实数上的周期函数

**定义 16（实数上的周期函数）**
$f: \mathbb{R} \to V$ 满足 $f(t+T)=f(t)$。一般形式：$f(t)=\sum c_n e^{i 2\pi n t/T}$，即 $U(1)$ 的不可约表示分解。

### 5.5 复数上的周期函数

**定义 17（复数上的椭圆周期函数）**
$f(z+\omega_1)=f(z+\omega_2)=f(z)$，$\omega_1,\omega_2$ 为 $\mathbb{R}$-线性无关，对应椭圆函数。

**定义 18（复数上的旋转周期函数）**
$f(qz)=f(z)$，$q=e^{i\theta}$，$\theta/2\pi \in \mathbb{Q}$，周期群为有限循环群。

### 5.6 四元数上的周期函数

**定义 19（四元数上的旋转周期函数）**
$f(qxq^{-1})=f(x)$，$q$ 为单位四元数，周期群为 $\langle q \rangle \subset SU(2)$。一般形式由 Peter-Weyl 定理给出：
$$f(x) = \sum_{j=0}^{\infty} \sum_{m,n=-j}^{j} c_{mn}^j \, D_{mn}^j(x),$$
其中 $D_{mn}^j$ 是 $SU(2)$ 的 $j$ 阶 Wigner D-矩阵元素。

## 第七部分：梯度、散度、旋度与四元数麦克斯韦方程组

### 7.1 梯度、散度、旋度

梯度 $\nabla\phi = (\partial_{x_1}\phi,\dots,\partial_{x_n}\phi)$ 指向标量场增加最快的方向。散度 $\nabla\cdot\mathbf{F} = \sum \partial_i F_i$ 度量向量场的源汇强度。旋度 $\nabla\times\mathbf{F}$（三维为向量，二维退化为标量）度量向量场的局部旋转强度。

> **解释性备注（非数学定理）**：在周期数学的解释框架中，梯度与 0 对称关联（势差驱动位移），散度与 1 对称关联（源汇比值定义缩放），旋度与 $i$ 对称关联（二维旋度对应复相位旋转，三维旋度对应四元数虚部代数）。

### 7.2 四元数梯度算子与麦克斯韦方程组

引入四元数梯度算子 $\nabla = i\partial_x + j\partial_y + k\partial_z$。对于四元数场 $F = \mathbf{E} + i\mathbf{B}$，麦克斯韦方程组可写为紧凑的四元数形式：
$$\nabla F = \rho - \mathbf{J},$$
其中 $\rho$ 为电荷密度，$\mathbf{J}$ 为电流密度。展开后，实部给出 $\nabla\cdot\mathbf{E} = \rho$（高斯定律），虚部给出 $-\nabla\times\mathbf{B} + \frac{\partial\mathbf{E}}{\partial t} = -\mathbf{J}$（安培-麦克斯韦定律）。这一表述将梯度、散度、旋度统一在四元数乘法的非交换结构中。

### 7.3 算子与对称性中心的关联

| 算子 | 场论作用 | 四元数表达 | 对称性解释 |
| :--- | :--- | :--- | :--- |
| **梯度** | 势差驱动 | $\nabla\phi$（标量场） | 0 对称（位移方向） |
| **散度** | 源汇度量 | $\operatorname{Re}(\nabla F)$ 标量部分 | 1 对称（缩放方向） |
| **旋度** | 旋转度量 | $\operatorname{Im}(\nabla F)$ 向量部分 | $i$ 对称（相位旋转） |

> 此关联属于周期数学框架内的理论解释，非已有数学定理。

## 第八部分：波与频谱——基于范畴化操作的计算

波的传播与频谱分析是周期数学的核心应用之一。本章从核函数出发，将所有频谱操作拆分为五个彼此独立的范畴：群作用（一元变换）、相互作用（多元耦合）、表示变换（基底变换）、域变换（定义域改变）、投影与选择（降维与压缩）。这五个范畴穷尽了周期数学中对频率、相位、周期和振幅的所有基本操作。

### 8.1 核函数

核函数描述了局部扰动如何随时间在空间中传播，或信号如何在变换域中被平滑与重建。在周期空间 $\mathfrak{P}_T$ 中，核函数可展开为不可约表示上的乘法算子：$K(x) = \sum_{\chi \in \widehat{G}} \hat{K}(\chi) \chi(x)$。

| 核函数 | 数学形式 | 物理/信息含义 | 对周期/频谱的影响 |
| :--- | :--- | :--- | :--- |
| **高斯核** | $e^{-x^2/2\sigma^2}$ | 热扩散、正态平滑 | 高频分量衰减，周期光滑化 |
| **指数核** | $e^{-\lambda\|x\|}$ | 放射性衰变、单边扩散 | 振幅沿轨道指数衰减 |
| **反距离核** | $1/\|x\|$ | 引力势、静电势 | 远程关联，长程有序 |
| **柯西核** | $1/(1+x^2)$ | 洛伦兹线型、重尾分布 | 保持长程关联，异常值鲁棒 |
| **三角核** | $1-\|x\|$（紧支撑） | 线性插值、局部平均 | 局部平滑，细节保留 |
| **Sinc 核** | $\sin(x)/x$ | 理想低通滤波 | 频域矩形窗，完全去除高频 |
| **贝塞尔核** | $J_0(x)$ | 柱面波、径向扩散 | 适用于圆形或柱对称周期域 |
| **拉普拉斯核** | $e^{-\sqrt{x^2}}$ | 边缘检测、突变捕捉 | 周期边界锐化 |
| **双曲正割核** | $\operatorname{sech}(x)$ | 孤子型传播 | 保持特定波形不变 |
| **墨西哥帽核** | $(1-x^2)e^{-x^2/2}$ | 边缘增强（高斯拉普拉斯） | 提取周期突变点 |

### 8.2 频谱基础操作的范畴化分类

#### 范畴一：群作用
保持表示空间不变的一元变换，是某个群在频谱函数空间上的作用。

| 群作用 | 群 | 数学定义 | 物理意义 |
| :--- | :--- | :--- | :--- |
| **频率平移** | $(\mathbb{R},+)$ | $(T_a \hat{\psi})(\omega) = \hat{\psi}(\omega - a)$ | 调制，频谱整体搬移 |
| **频率缩放** | $(\mathbb{R}^+,\times)$ | $(S_\lambda \hat{\psi})(\omega) = \hat{\psi}(\omega/\lambda)$ | 时间压缩/展宽 |
| **相位旋转** | $U(1)$ | $(R_\phi \hat{\psi})(\omega) = e^{i\phi} \hat{\psi}(\omega)$ | 整体相移，时延 |
| **振幅增益** | $C(\Omega,\mathbb{R}^+)$ | $(A_g \hat{\psi})(\omega) = g(\omega)\hat{\psi}(\omega)$ | 滤波，选频放大/衰减 |
| **频率轴反射** | $C_2$ | $(M \hat{\psi})(\omega) = \hat{\psi}(-\omega)$ | 频谱反转，时域复共轭 |

#### 范畴二：相互作用
两个（或多个）周期结构的二元或多元运算，产生新的频率分量。这是周期数学区别于传统调和分析的特色方向。

| 相互作用 | 数学定义 | 物理意义 | 产生的新周期 |
| :--- | :--- | :--- | :--- |
| **线性叠加** | $(\hat{\psi}_1 + \hat{\psi}_2)(\omega)$ | 干涉 | 无新频率，仅振幅/相位变化 |
| **频域卷积** | $(\hat{\psi}_1 * \hat{\psi}_2)(\omega)$ | 混频，时域乘积 | 和频 $\omega_1+\omega_2$，差频 $\|\omega_1-\omega_2\|$ |
| **点乘** | $(\hat{\psi}_1 \cdot \hat{\psi}_2)(\omega)$ | 频域加窗，时域卷积 | 频率选通 |
| **内积/相关** | $\langle \hat{\psi}_1, \hat{\psi}_2 \rangle$ | 匹配检测，相干性度量 | 匹配度标量 |

#### 范畴三：表示变换
改变信号的表示基底，将一个希尔伯特空间中的信号映射到另一个希尔伯特空间。

| 表示变换 | 原空间 | 目标空间 | 核心性质 |
| :--- | :--- | :--- | :--- |
| **傅里叶变换** | $L^2(\mathbb{R}, dt)$ | $L^2(\mathbb{R}, d\omega)$ | 对角化平移群作用 |
| **短时傅里叶变换** | $L^2(\mathbb{R})$ | $L^2(\mathbb{R}^2)$ | 局部频率分析 |
| **小波变换** | $L^2(\mathbb{R})$ | $L^2(\mathbb{R} \times \mathbb{R}^+)$ | 对角化仿射群作用 |
| **拉普拉斯变换** | 时域函数 | 复频域函数 | 增长/衰减模式分解 |
| **Z 变换** | 离散序列 | 复平面单位圆 | 离散系统的频率响应 |
| **球谐展开** | $L^2(S^2)$ | 球谐系数序列 | 对角化 $SO(3)$ 旋转群 |

#### 范畴四：域变换
改变信号所依赖的变量集合本身——从连续到离散，从均匀到非均匀。

| 域变换 | 数学定义 | 物理意义 |
| :--- | :--- | :--- |
| **采样** | $\psi[n] = \psi(n\Delta t)$ | 连续→离散，限制到 $\Delta t \mathbb{Z}$ |
| **插值/重建** | $\psi(t) = \sum_n \psi[n] \operatorname{sinc}(t-n\Delta t)$ | 离散→连续 |
| **重采样** | 改变 $\Delta t$（抽取/内插） | 改变离散子群的步长 |
| **周期性延拓** | $\psi_{\text{per}}(t) = \sum_k \psi(t - kT)$ | 从非周期到周期 |

> **Pontryagin 对偶视角**：采样是从连续群 $\mathbb{R}$ 到其离散子群 $\Delta t\mathbb{Z}$ 的限制，傅里叶变换将此限制映射为对偶群 $U(1)$ 上的周期性延拓——这是采样定理的群论本质。

#### 范畴五：投影与选择
从完整频谱中选择一个子集，或将高维表示投影到低维子空间。

| 投影/选择 | 数学定义 | 物理意义 |
| :--- | :--- | :--- |
| **带通滤波** | $\hat{\psi}' = \hat{\psi} \cdot \mathbf{1}_{[\omega_1,\omega_2]}$ | 选择特定频段 |
| **降采样（抽取）** | 保留第 $M$ 个样本，丢弃其余 | 降低数据率 |
| **主成分分析** | 投影到最大方差方向 | 降维，提取主导模式 |
| **阈值压缩** | 保留 $\|\hat{\psi}\|>\tau$ 的分量，其余置零 | 稀疏化 |

### 8.3 从基础操作到复杂操作的构建

| 复杂操作 | 构建方式 | 涉及的范畴 |
| :--- | :--- | :--- |
| **平移** | 频率依赖的相位旋转 | 群作用（相位旋转） |
| **缩放** | 频率缩放的逆 | 群作用（频率缩放） |
| **过滤** | 振幅增益 $g(\omega)$ | 群作用（振幅增益） |
| **卷积** | 频域乘积 | 群作用（振幅增益 + 相位旋转） |
| **扩散** | 乘以高斯传递函数 | 群作用（振幅增益） |
| **扰动** | 加性源 | 相互作用（线性叠加） |
| **采样** | 时域采样 ↔ 频域周期延拓 | 域变换（采样）+ 群作用 |
| **重采样** | 插值 + 抽取 | 域变换 + 群作用（振幅增益） |
| **变换** | 基变换 | 表示变换 |
| **基频/拍频** | 差频的产生 | 相互作用（频域卷积） |
| **共振** | 增益峰 + 和频 | 群作用（振幅增益）+ 相互作用 |
| **耦合** | 乘积项 ↔ 频域卷积 | 相互作用（频域卷积） |
| **抑制** | 陷波滤波 | 群作用（振幅增益） |
| **锁定/解锁** | 相位反馈 + 非线性吸引子 | 群作用（相位旋转）+ 相互作用 |
| **能量/分贝** | $\int \|\hat{\psi}\|^2 d\omega$ | 群作用（振幅增益）的度量 |
| **熵** | $-\sum p_i \log p_i$ | 投影（概率化）+ 度量 |
| **压缩** | 变换 + 阈值 + 编码 | 表示变换 + 投影（阈值） |
| **频率聚类** | 频谱上的距离度量 + 分组 | 群作用（频率平移）+ 度量 |
| **频分复用** | 多路信号调制到不同频段 | 群作用（频率平移） |

**周期数学的核心视角**：以上所有操作最终都可归结为对频率、相位、周期和振幅在五个基本范畴下的组合。其中“相互作用”范畴是周期数学区别于传统调和分析的特色所在——它研究不同周期之间如何通过耦合产生新周期、共振、锁定与湮灭。

## 第九部分：扰动、耦合与干涉

扰动是周期系统动态演化的基本驱动力。

### 9.1 点扰动

**定义 20（点扰动）**
在位置 $x_0$ 注入的扰动：$\psi'(x) = \psi(x) + \varepsilon \cdot \delta(x - x_0)$，$\varepsilon \in \mathbb{C}$ 为强度。

### 9.2 干涉

两个扰动叠加时产生干涉项：$|\psi_1+\psi_2|^2 = |\psi_1|^2+|\psi_2|^2+2|\psi_1||\psi_2|\cos(\Delta\phi)$。同相增强，反相抵消。

### 9.3 扰动对周期结构的影响

| 影响维度 | 数学刻画 | 物理表现 |
| :--- | :--- | :--- |
| **周期变化** | 周期群变更为子群或商群 | 节律重置 |
| **振幅变化** | 扩散核平滑或锐化 | 能量衰减或增强 |
| **频率变化** | 新频率分量被激发 | 频谱展宽 |

## 总结

周期数学以对称性与方向性为根基，以数系扩张中对称群的递升为主线，以群作用轨道分解定义周期性，为每一种数系建立了对应的一般周期函数。梯度、散度、旋度通过四元数麦克斯韦方程组获得统一表达，并与 0、1、$i$ 三大对称性中心形成解释性关联。波与频谱的操作由五个基本范畴统一描述——群作用、相互作用、表示变换、域变换、投影与选择——其中相互作用是周期数学区别于传统调和分析的特色方向，聚焦于不同周期之间如何生成新周期、共振、锁定与湮灭。全框架接受现代数学的全部标准构造，目标在于为周期现象提供统一的数学语言。
